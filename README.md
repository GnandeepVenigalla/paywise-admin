# Paywise Admin Dashboard

A professional, high-performance administrative panel for Paywise.

## Tech Stack
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS 4
- **Charts**: Recharts
- **Icons**: Lucide React
- **API**: Axios

## Features
- **Dashboard**: Real-time summary of users, expenses, and groups.
- **Data Analysis**: Visual representation of user growth and transaction volume.
- **User Management**: View and manage the entire user base.
- **Advanced Stats**: Demographic and device distribution analytics.
- **System Settings**: Configure global application parameters.

## Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`
